# 2026-07-15 Brief 项目架构重构

## 重构内容
1. **Clean URL 支持**：render.py 输出从 `{name}.html` 改为 `{name}/index.html`，访问 `/dsh` 替代 `/dsh.html`
2. **base_path 机制**：模板添加 `{{ base_path }}` 变量和 `url` Jinja2 过滤器，处理子目录层级资源路径（`../assets/` vs `./assets/`）
3. **GitHub Actions CI**：`.github/workflows/static.yml` 添加 Python 构建步骤，push JSON 即自动渲染部署
4. **生成文件不入库**：.gitignore 忽略 `index.html`、`dsh/`、`cyy/` 等生成产物
5. **Bug 修复**：template.html 中 home social links 区域现在正确识别 `fa` 前缀的 Font Awesome 图标
6. **新增 requirements.txt**

## 关键设计决策
- 保留 Jinja2 SSG 方案（SEO 友好、加载快），通过 CI 自动化解决编辑便利性问题
- 根 index.html = 默认简历(dsh)，同时 /dsh/ 也可访问（base_path 不同但内容相同）
- url 过滤器自动处理外部 URL(http/https)、绝对路径(/)、相对路径(assets/...)
